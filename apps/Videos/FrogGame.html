Frog Game/Controller Ad

1st September 2022


This is a video a friend and I created as part of a school assignment, where we had to showcase a controller and game we created. The game is a remake of the classic game Frogger. When it came to the controller, we were pretty much free to make whatever we wanted. Using an Arduino, we created a controller that required the user to touch water to control the frog.
